# Cruces-fronterizos

Cruces Fronterizos México–USA mediante Simulación en ProModel

Este proyecto modela y analiza el flujo vehicular en tres cruces fronterizos estratégicos entre México y Estados Unidos utilizando simulación de eventos discretos en ProModel. El objetivo principal es evaluar cómo la configuración de carriles, casetas y entradas influye en la productividad, los tiempos de espera y el desempeño general del sistema.

Objetivo del Proyecto

Evaluar el impacto de distintas configuraciones operativas en cruces fronterizos mediante un modelo de simulación que permita:

Medir tiempos de espera.

Analizar la utilización de casetas.

Identificar cuellos de botella.

Proponer mejoras sin necesidad de intervenir físicamente la infraestructura.

Cruces Fronterizos Analizados

Colombia–Solidarity (Laredo)

Otay Mesa

Ysleta / El Paso

Cada cruce cuenta con un modelo independiente y escenarios ajustados según su infraestructura real.
